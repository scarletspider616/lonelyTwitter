Elastic search 
	schema free (no sql) 
	open source (apache) 
	uses RESTful API 
Who uses it? 
	netflix
	github
	stack overflow
	rogers
	ebay
How does it work? 
	RESTful API
		REpresentational State Transfer
			protocol doesn't assume state is stored through communications
			stateless
		operates over HTTP
		POST/PUT = Create
			POST - submitting a request
			PUT - submitting a request and updating 
		GET = Read
		POST/PUT = Update
		DELETE = delete
Some useful tools
	Linux - CURL 
		see CURL tutorial in this dir
	Windows - Fiddler
	web - RESTClient
